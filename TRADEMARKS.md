**Purser** is the name and brand of this project. The software is licensed under
**Apache-2.0** (see [`LICENSE`](LICENSE)), which grants broad rights to use,
modify, and redistribute the **code** — but, per Apache-2.0 §6, it grants **no
rights to the project's names or logos**. This file states how the marks may be
used.

## The marks
- The word mark **"Purser"** (as the name of this software).
- The **Purser logo** — the clearance/load-line seal in
  [`assets/brand/`](assets/brand/).

*(Registration status: the mark is not yet registered; a clearance search and

## What you may do (no permission needed)
- Use the code under Apache-2.0.
- Use the name **nominatively** — to truthfully refer to this project (e.g.
  "compatible with Purser", "scanned with Purser", blog posts, comparisons).
- Keep the name/notices intact when redistributing unmodified builds.

## What needs permission / is not allowed
- **Don't** use "Purser" or the logo as the name of **your** product, fork,
  service, or company, or in a way that implies endorsement or official status.
- **Don't** register "Purser" (or a confusingly similar name) or the logo as a
  trademark, domain, package, or social handle.
- **Rename hard forks.** If you distribute a modified version, use your own name
  so users aren't confused about origin.
- **Don't** alter the logo (recolor/distort) and keep calling it the Purser logo.
